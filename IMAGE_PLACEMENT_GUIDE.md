## Image Specifications

### Hero Image (Priority 1)
```
Filename: product-hero.jpg
Size: 800x600px (or larger, same ratio)
Format: JPG (optimized) or WebP
File Size: < 500KB
Background: White or light colored
Quality: High (85-90%)
```

### Showcase Image (Priority 2)
```
Filename: product-showcase.jpg
Size: 600x500px (or larger, same ratio)
Format: JPG (optimized) or WebP
File Size: < 400KB
Background: Natural or lifestyle setting
Quality: High (85-90%)
```

### Optional Images (Priority 3)
```
Filenames: product-detail-1.jpg, product-detail-2.jpg, etc.
Size: 400x400px or larger
Format: JPG or PNG
File Size: < 300KB each
```

## Photography Tips

### Lighting:
- âœ… Natural daylight (near window)
- âœ… Soft, diffused light
- âœ… Avoid harsh shadows
- âŒ No direct flash
- âŒ No yellow/warm indoor lighting

### Background:
- âœ… Clean white or light gray
- âœ… Neutral colors
- âœ… Uncluttered
- âŒ Busy patterns
- âŒ Distracting objects

### Angles:
- âœ… Eye-level or slightly above
- âœ… Show product clearly
- âœ… Multiple angles for gallery
- âŒ Extreme angles
- âŒ Distorted perspectives

### Composition:
- âœ… Product fills 70-80% of frame
- âœ… Centered or rule of thirds
- âœ… Sharp focus
- âŒ Too much empty space
- âŒ Product too small in frame

## Image Editing Checklist

Before uploading, ensure:
- [ ] Image is in focus and sharp
- [ ] Colors look natural

- [ ] File is named correctly
- [ ] Alt text is descriptive

## Image Optimization Tools

### Online (Free):
- **TinyPNG**: https://tinypng.com
- **Squoosh**: https://squoosh.app
- **Compressor.io**: https://compressor.io

### Desktop:
- **Photoshop**: Save for Web
- **GIMP**: Export as JPG (85% quality)
- **XnConvert**: Batch processing

### Command Line:
```bash
# ImageMagick
magick input.jpg -resize 800x600 -quality 85 output.jpg

# WebP conversion
cwebp -q 85 input.jpg -o output.webp
```

## Need Help?

### Can't take good photos?
- Use smartphone in good lighting
- Follow DIY setup guide above
- Or hire local photographer

### Images too large?
- Use TinyPNG to compress
- Resize to recommended dimensions
- Convert to WebP format

### Not sure which image to use?
- Test multiple options
- Ask friends/family for feedback
- Use the clearest, most professional looking one
